# Juego del Ahorcado - Alura One

🎮 Un clásico juego del ahorcado interactivo desarrollado como proyecto de práctica con herramientas de IA.

## Descripción

Este proyecto es una versión renovada del clásico juego del ahorcado, desarrollado como parte de un ejercicio para explorar y aprender nuevas tecnologías de desarrollo web, con la asistencia de herramientas de Inteligencia Artificial.

## Características

- 🎯 **Tres niveles de dificultad**: Fácil, Normal y Difícil
- 🎨 **Diseño moderno** con modo claro/oscuro
- 🔊 **Efectos de sonido** generados con Web Audio API
- 💾 **Estadísticas persistentes** guardadas en localStorage
- 💡 **Sistema de pistas** según la dificultad
- ⌨️ **Teclado virtual** y soporte para teclado físico
- 📱 **Diseño responsivo** adaptable a dispositivos móviles
- ♿ **Accesibilidad** mejorada con atributos ARIA

## Tecnologías Utilizadas

- **Vanilla JavaScript** - Versión clásica
- **Vue 3 + Vite** - Versión moderna (en desarrollo)
- **CSS3** con variables y animaciones
- **Web Audio API** para sonidos
- **localStorage** para persistencia de datos

## Cómo Ejecutar

### Versión Clásica (funcional)

Simplemente abre el archivo `index.html` en tu navegador:

```
ahorcadoAluraOne-main/index.html
```

O si prefieres usar un servidor:

```bash
cd ahorcadoAluraOne-main
npx serve
```

### Versión Vue 3 (en desarrollo)

```bash
# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Build para producción
npm run build
```
